# Php web server

##  General
* You can swap php versions: from 8.1 => 7.4
* You can modify your nginx.conf in .docker/nginx/nginx.conf
* Database: MySQL
* For database management: Adminer

## Ports
* Nginx: 8000
* Adminer: 8080 // login: root, password:root
* MySQL: 3306
